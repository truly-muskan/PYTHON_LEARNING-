# Python Learning Assignment

## Overview

This repository contains Python programs created while learning the fundamentals of Python programming. The purpose of this repository is to demonstrate understanding of basic Python concepts through multiple examples and practice programs.

### 1. Variables and Data Types

* Integer
* Float
* String
* Boolean
* Type Checking

### 2. Operators

* Arithmetic Operators
* Comparison Operators
* Assignment Operators

### 3. Conditional Statements

* if
* if-else
* if-elif-else
* Nested Conditions

### 4. Loops

* for Loop
* while Loop
* Loop Control Statements

### 5. Functions

* Function Definition
* Function Parameters
* Return Values
* Function Calls

### 6. Lists

* Creating Lists
* Accessing Elements
* Updating Lists
* List Methods

### 7. Tuples

* Creating Tuples
* Accessing Elements
* Tuple Operations

### 8. Dictionaries

* Key-Value Pairs
* Accessing Values
* Updating Dictionaries
* Dictionary Methods

### 9. Sets

* Creating Sets
* Adding and Removing Elements
* Set Operations

### 10. Object-Oriented Programming (OOP)

* Classes
* Objects
* Constructors
* Methods

### 11. Inheritance

* Parent Class
* Child Class
* Reusing Code Through Inheritance

### 12. Polymorphism

* Method Overriding
* Different Behaviors Using the Same Method Name
